1 - Type: integer, Val: 32

CODE:

LD_LOCAL_C 1 1 ; Load local variable 1 with constant 1
PUSH_LOCAL 0 ; Push the value of local variable 0 to the stack
PUSH_LOCAL 1 ; Push the value of local variable 1 to the stack
STR_CAT ; Concatonate the two values at the top of the stack
LD_LOCAL_STACK 2; Load local variable 2 with the value at the top of the stack
